# Mobil Usta Site Ozeti

Bu proje, mobil oto tamir hizmeti tanitimi icin hazirlanmis responsive bir web sitesidir.

## Proje Amaci
- Kullaniciya hizli sekilde hizmetleri gostermek
- WhatsApp uzerinden kolayca bilgi ve randevu talebi almak
- Mobil, tablet ve masaustu ekranlarda sorunsuz calismak

## Ana Ozellikler
- Tek sayfa akisi (index.html):
  - Ana Sayfa
  - Hizmetlerimiz
  - Nasil Calisir
  - SSS
  - Iletisim
- Sticky navbar ve mobil hamburger menu
- SSS bolumunde acilir/kapanir soru kartlari
- WhatsApp hizli iletisim butonlari
- Responsive tasarim (1024px, 768px, 480px kirilimlari)

## Mevcut Dosyalar
- index.html: Tek sayfa ana yapi ve tum bolumler
- services.html: Hizmetler sayfasi
- how-works.html: Nasil calisir sayfasi
- faq.html: SSS sayfasi
- contact.html: Iletisim sayfasi
- mobil.html: Onceki surumden kalan dosya
